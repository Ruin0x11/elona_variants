# Elona Variants

This page will attempt to collect the major features of each Elona variant in one place. This is to better understand what features across all the variants could be ported OpenNefia through its mod system.

Because each variant has so many changes, it's probably impractical to list them all here. For now I'll focus on the features that seem to have the most importance to people.

Each variant can also have a mod identifier associated with it. In OpenNefia, we could implement "modpacks" by simply creating an empty mod that specifies a bunch of related mods as dependencies.

This page was autogenerated from data at the repository [Ruin0x11/elona\_variants](https://github.com/Ruin0x11/elona_variants). The dates are current as of 07/24/2020. If there's a feature you want to see a mod created for, feel free to open a pull request that adds it there.
